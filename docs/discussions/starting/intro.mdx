# Starting a Discussion

> **Discussion:** _The action or process of talking about something in order to reach a decision or to exchange ideas._

A **Discussion** should be about "_exchnaging ideas_". If you just want to send a quick email, use the [Group Email](/docs/started/Group%20Actions/intro#send-an-email) on the **Group Home Page**.

<img src={useBaseUrl('/img/discussions/startDiscussion.gif')} />

### Setting an End Date

No one wants an endless Discussion, so **LORROS** allows you to set the **End Date** for each Discussion _(the default is 7 days)_.

:::info Info
When the Discussion has ended _(archived)_, the Group Admin can [reset the End Date](/administration/discussions/intro).
:::

<img src={useBaseUrl('/img/discussions/discussionDate.gif')} />

### Adding Files

Once you have entered the Discussion **Subject** and **Content**, you will be able to add **Files** from the **ADD FILE** tab.

<img src={useBaseUrl('/img/global/addFiles.gif')} />

### Adding Polls

Adding **Polls** to Discussions are a great way to find common ground and keep the Discussion moving in a productive direction. **LORROS** makes it easy to add polls at any time. Learn how to [start a **Poll**](../../misc/Start%20a%20Poll/intro)

### Submit the Discussion

When you **Submit** the **Discussion**, an email is sent to all the Group Members ... **Let the Discussion Begin!**

## Discussion Heading Tabs

At the top of the **Discussion**, you will find a tab for **Files** _(if there are any Files)
_ and the **Print** tab.

### Files

All **Files** for the Discussion can be accessed from the Files tab.

<img src={useBaseUrl('/img/global/openFiles.gif')} />

### Print

You can print out the Discussion at any time using the **Print** tab at the top of the Discussion.

<img src={useBaseUrl('/img/global/print.gif')} />
